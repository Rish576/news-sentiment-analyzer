# News Sentiment Analyzer

This project analyzes the sentiment of real-time news articles using Python and NLP.

## Features
- Fetches live news using NewsAPI
- Preprocesses text using NLTK
- Performs sentiment analysis using TextBlob
- Visualizes frequent words using WordCloud

## Technologies Used
- Python
- NLTK
- TextBlob
- NewsAPI
- WordCloud
